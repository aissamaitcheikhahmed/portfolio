Rubric System - Hogent Academic Project

This project was developed as part of my Hogent opleiding, in collaboration with my coordinator. It is a well-structured, secure academic evaluation platform built with a .NET C# backend (using SQL Server, Entity Framework, JWT authentication, and MVC architecture) and a modern React TypeScript frontend. The system is designed for maintainability, scalability, and robust role-based access control.

Login & Dashboard

  • Users can log in and see a role-based dashboard.
  • Sidebar navigation to Rubrics, Evaluations, and more.

Rubric Management

  • Create RubricTemplates (with title and description).
  • Create Rubrics based on templates (linked to Course & AcademicPeriod).
  • List view of all Rubrics.
  • On selecting a Rubric: see linked CriteriumTemplates and add new ones.

Evaluations

  • "My Evaluations" page shows evaluations assigned to the logged-in user.
  • Each evaluation displays key info (planned/presentation date, etc.).
  • Button to open and fill in the evaluation.

Evaluation Uitvoeren

  • Opening an evaluation shows all criteria from the RubricTemplate.
  • For each criterion: available ScoreTemplates (Excellent, Good, Fair), remarks field.
  • User selects a score and adds remarks; data is saved securely.

Project Note

I worked on this project with a strong focus on clean architecture, security (JWT), and best practices in both backend (.NET C#, SQL Server, Entity Framework, MVC) and frontend (React, TypeScript). The result is a robust, maintainable, and secure academic evaluation platform.
